#  Workout Planner

## Description

A Python-based CLI app that calculates BMI from weight and height inputs, then provides:
- Daily workout time
- Basic diet suggestion
- Workout location (home/gym)


## BMI Categories
| Category        | BMI Range     |
|----------------|---------------|
| Underweight     | < 18.5        |
| Normal          | 18.5–24.9     |
| Overweight      | 25.0–29.9     |
| Obese           | 30.0+         |
